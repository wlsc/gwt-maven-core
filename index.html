GWT Maven core project

The base for further GWT project development

Download as .zip Download as .tar.gz View on GitHub

Base for your next Maven-based GWT project

Checkout and start your next pure GWT project without a need to configure everything by yourself.

Package Contains

  • Modular web app
  • Maven
  • Thomas Broyer's GWT Maven Plugin
  • MVP pattern
  • Activities and Places
  • Localization
  • GSS (CSS) handling
  • Packaging to WAR shared run config

Usage (Eclipse)

  • Download and install gwt-eclipse-plugin
  • Import this project into workspace
  • Compile the client "project-client" with run as -> GWT Compiler
  • For Jetty development follow this guide, for Tomcat development follow these guilde for manual and automatic SuperDevMode deployment (thanks Brandon!).
